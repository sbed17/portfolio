# Project 5 – WineMag Report

**Tools:** Power BI, DAX, PowerQuery

**Description:**  
Interactive report offering insights about wine bottles spanning 6 sections: Overview, Wine Insights, Geographical Insights, Wineries, Our Tasters and Select Your Wine.
